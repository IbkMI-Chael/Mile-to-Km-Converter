# Mile to Km Converter

A simple desktop app that converts miles to kilometres 
instantly using a clean GUI built with Python.

## Technologies Used
- Python 3
- Tkinter (built-in Python GUI library)

## How to Run It
1. Make sure you have Python 3 installed
2. Download the file:
   - miles_to_km_converter.py
3. Open your terminal in that folder
4. Run: python miles_to_km_converter.py

## How to Use It
- Type a distance in **Miles** into the input box
- Click **Calculate**
- The result in **Kilometres** appears instantly

## Features
- Converts miles to kilometres in real time
- Clean and simple GUI
- Lightweight — runs on any computer with Python installed
